# Quran Verses - Custom CSS Animations

## Description
This project is a minimalist, single-page web interface designed to showcase bilingual Quranic verses (Arabic and English). The core technical challenge of this task focused on mastering web typography, layout distribution, and custom CSS animations to achieve fluid visual transitions without relying on JavaScript animation libraries.

## Live Demo
Explore the live web application here: (https://nadaalafify.github.io/Quran-Verses-CSS-Animations/)

## Tech Stack
- **HTML5:** Structuring text blocks, localized translation containers, and semantic card wrappers.
- **CSS3 (Pure CSS):** Leveraging advanced CSS Keyframe animations, custom transition timing functions, layout centering, and smooth box-shadow mechanics.

## Key Features and Technical Execution
- **Bilingual Typographic Layout:** Symmetrical alignment separating traditional Arabic calligraphy from standard English translation blocks inside a clean user interface.
- **Custom CSS Keyframes:** Smooth fade-in, sliding, or scaling animation loops managed entirely through raw CSS properties to maximize rendering performance.
- **Cohesive Color Palette:** Implementation of soft, muted pastel tones tailored for reading comfort and minimal visual clutter.
- **Frameworkless Architecture:** Developed 100% manually to reinforce underlying knowledge of animation curves, delays, and state management in vanilla CSS.
